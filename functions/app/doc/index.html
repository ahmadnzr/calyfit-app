Calyfit public API

URL https://asia-southeast2-calyfit-app.cloudfunctions.net/app

and below is the endpoint :

[1] Users

GET
  • get all user : /api/users/
  • get user by id: /api/users/{id}

example:
curl -X GET -H "Content-type: application/json" "https://asia-southeast2-calyfit-app.cloudfunctions.net/app/api/users"

POST
  • post new user: /api/users/
UPDATE
  • update exiting user: /api/users/{id}
DELETE
  • delete exiting user: /api/users/{id}
